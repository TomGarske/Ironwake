# Replicants

## Subtitle
Replicants: Swarm Command

## Game Type
Strategy expansion / base growth (PVE)

## Player Count
1 to max players

## Objective
Harvest metal, expand the swarm, and execute replication protocols to dominate the map.

## Core Concept
This mode tells the origin story of the Replicant threat. Players command replicator entities inspired by Stargate SG-1, balancing resource intake, growth, and coordinated expansion under rising environmental pressure.

## Notes
- All sessions are player-versus-environment.
- Focus is strategic scaling and autonomous swarm behavior.
