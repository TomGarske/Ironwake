# Blacksite Breakout

## Subtitle
Escape from Area 51

## Game Type
Tactical RPG-style escape sim (PVE)

## Player Count
1 to max players

## Objective
Break out of a compromised Area 51 containment zone as experimental entities with unique powers.

## Core Concept
Containment has failed. Each player controls a different sci-fi creature class in a Fallout 2-style interface. Runs begin in a single room with fog of war, and the map reveals as players explore procedurally generated sectors toward escape.

## Current Creature Options
1. Replicators (Stargate-inspired)
2. Fungus strain (The Last of Us-inspired)
3. CRISPR gene-therapy anomaly
4. Rogue AI construct

## Naming Direction
Monster names should be gender-neutral (example: "Chris" for a CRISPR-derived entity).

## Notes
- All sessions are player-versus-environment.
- Heavy emphasis on asymmetrical abilities and replayable map generation.
