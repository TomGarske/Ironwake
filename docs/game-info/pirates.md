# Pirates

## Subtitle
Void Corsairs

## Game Type
Naval combat action (PVE)

## Player Count
1 to max players

## Objective
Choose cruiser and ship type, then fight enemy fleets in simple naval combat encounters.

## Core Concept
`Pirates` is the current base game mode. Players sail into combat scenarios, select ship loadouts, and battle AI enemies. Story integration will be expanded later.

## Notes
- All sessions are player-versus-environment.
- This mode is the current default experience.
