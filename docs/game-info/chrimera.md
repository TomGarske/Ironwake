# Chrimera

## Subtitle
Chrimera: Bioforge Run

## Game Type
Side-scroller roguelike (PVE)

## Player Count
1 to max players

## Objective
Escape from an underground Area 51 research complex while a CRISPR outbreak creates hostile monsters.

## Core Concept
Players are scientists using experimental tools to survive and escape. Each level represents a deeper underground floor segment. Teams push upward through escalating threats, adapting run-to-run with roguelike pacing.

## Notes
- All sessions are player-versus-environment.
- Setting and tone connect directly to the Blacksite Breakout universe.
